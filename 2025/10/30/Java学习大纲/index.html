Java学习大纲

🧭 Java 学习大纲

阶段 学习主题 主要内容 学习目标
阶段一 Java 基础语法 - Java 平台与 JVM、JRE、JDK 概念
- 基本数据类型与运算符
- 控制语句（if, switch, for, while）
- 类与对象、构造方法、方法重载
- 封装、继承、多态
- 抽象类与接口
- 内部类与匿名类
- String 与日期时间类		 能编写结构清晰、面向对象的 Java 程序
阶段二 集合、异常与 I/O - 异常体系与自定义异常
- 集合框架：List, Set, Map, Queue
- 泛型（Generics）
- 迭代器与 Stream API
- 文件 I/O：File, Reader/Writer, NIO
- 对象序列化与反序列化		 掌握数据结构与文件操作的基础技能
阶段三 多线程与并发编程 - Thread 与 Runnable
- 同步与锁（synchronized, ReentrantLock）
- 线程池（ExecutorService）
- 并发集合（ConcurrentHashMap）
- CompletableFuture 与异步编程
- 原子类与并发包		 理解并发机制与线程安全设计
阶段四 JVM 与内存机制 - JVM 架构与类加载机制
- 内存模型（堆、栈、方法区）
- 垃圾回收与调优参数
- JIT 编译与性能分析工具（jconsole, VisualVM）		 理解 Java 程序运行原理与性能优化
阶段五 标准库与网络编程 - 日志框架（Log4j, SLF4J）
- 网络通信（Socket, HTTPClient）
- JSON/XML 解析（Jackson, Gson, JAXB）
- JDBC 数据库操作
- 单元测试（JUnit, Mockito）		 掌握 Java 应用开发的常用模块
阶段六 Spring Boot 框架入门 - IoC 与 AOP 核心思想
- Spring Boot 快速启动与项目结构
- RESTful API 开发
- 数据访问（Spring Data JPA / MyBatis）
- 配置管理（YAML, Profiles）
- 异常与日志处理
- 测试与打包（Maven/Gradle）		 掌握现代 Java 企业级开发标准
阶段七 项目实践与进阶 - 控制台项目：Todo 应用
- Web 项目：图书管理系统、账单管理系统
- 聊天服务器 / API 应用
- Web 开发：Spring Boot + Thymeleaf / React
- Docker、K8s 部署
- 微服务：Spring Cloud、API Gateway
- 安全：JWT、Spring Security		 能独立设计、实现、部署完整 Java 应用
阶段八 工程化与设计模式 - 构建工具：Maven / Gradle
- Git 版本控制
- CI/CD（GitHub Actions / Jenkins）
- 性能分析与调优
- 设计模式（23 种经典模式）
- Lambda、Stream、Optional		 掌握工程化与高质量代码开发能力
